# React Todo List

Udemyの講座内の課題として作成したToDoアプリです。  

## デモ

以下のURLから実際の動作を確認できます。
https://xzhong07.github.io/React_TodoList/

## 概要

このアプリケーションは、Reactを用いて作成したシンプルなToDoリスト管理アプリです。  
JavaScriptを使用して実装されており、コンポーネントベースの設計や状態管理の基礎を学ぶことを目的としています。

## 使用技術

- React
- JavaScript
- HTML / CSS

## 学習目的

- Reactにおけるコンポーネント設計の理解
- useState を用いた状態管理の基礎
- ユーザー操作（追加・削除・チェックなど）の実装
- UIの分割と再利用性の向上

## 備考

本プロジェクトはUdemyの学習教材をもとに作成したものです。
